# 🏢 AI OFFICE — 가상 AI 회사 (3D)

대표님이 한마디만 지시하면, 6명의 AI 임원 + PM 디렉터봇이 **3D 회의실에 모여 토론하고
결과물(보고서·제안서·양식·아이디어·조사자료)을 자동으로 만들어주는** 게임형 가상 사무실입니다.

## 🚀 시작하기

1. `ai_office_3d.html` 파일을 **더블클릭** (크롬/엣지 권장)
2. 상단 **▶ 데모 회의** 버튼으로 먼저 체험 (API 키 불필요)
3. 실제 AI 회의를 하려면:
   - [console.anthropic.com](https://console.anthropic.com/) 에서 API 키 발급
   - 상단 **⚙ 설정** → API 키 입력 → 저장
4. 아래 입력창에 지시 입력 → **지시 ▶**

> 더블클릭 실행이 안 되거나 화면이 비어 있으면 `start-server.bat`을 실행한 뒤
> 브라우저에서 `http://localhost:8787/ai_office_3d.html` 로 접속하세요.

## 🎮 조작법

- **마우스 드래그**: 카메라 회전 / **휠**: 줌 / **우클릭 드래그**: 이동
- **🛠 편집 모드**: 가구 클릭 후 드래그로 이동, `R` 키로 회전 (자동 저장)
- **📁 기록**: 지난 회의 결과물 다시 보기
- 결과물은 **Word / HTML / Markdown / PDF(인쇄)** 로 저장 가능

## ⚙ 비용 안내

- 회의 1회 ≈ API 호출 8회(1라운드) ~ 14회(2라운드)
- 설정에서 모델 변경 가능: Opus 4.8(최고 품질) / Sonnet 4.6(경제적) / Haiku 4.5(초저가)
- API 키는 이 PC의 브라우저에만 저장되며 외부로 전송되지 않습니다 (Anthropic API 호출 제외)
